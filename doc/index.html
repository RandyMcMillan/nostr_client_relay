Nostr

Nostr is a Internet protocol developed in 2019, that enables global, decentralized, and censorship-resistant social media.

Introduction to Nostr

Nostr_client_relay is a Nostr C++ engine that allows to build Nostr applications for command line, desktop or web. It is available on
